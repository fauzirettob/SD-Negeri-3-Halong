# SD Negeri 3 Halong - Website Resmi

Website resmi SD Negeri 3 Halong. Dibangun dengan HTML, CSS, dan JavaScript.

## Fitur

- 📰 **Berita & Informasi** - Update terbaru seputar sekolah
- 🏆 **Prestasi** - Galeri prestasi siswa
- 👨‍🏫 **Profil Guru** - Informasi tenaga pengajar
- 📸 **Galeri** - Dokumentasi kegiatan sekolah
- 📞 **Kontak** - Informasi kontak sekolah
- 🔐 **Admin Panel** - Panel manajemen konten

## Teknologi

- HTML5
- CSS3
- JavaScript
- JSON untuk penyimpanan data

## Cara Menjalankan

Buka `index.html` di browser atau kunjungi [GitHub Pages](https://fauzirettob.github.io/SD-Negeri-3-Halong/).
